# 🛒 Laravel Online Shop (E-Commerce Project)

##  Project Overview

This is a **Laravel-based E-Commerce Web Application** currently under development. The project is designed to build a complete online shopping system with both **Admin Panel** and **User Panel**.

At this stage, the **Admin Panel is fully functional**, while the **User Panel is under development**.

---

##  Features (Completed - Admin Panel)

###  Authentication

 Admin Login System
 Secure Dashboard Access

###  Dashboard

 Admin control panel overview
 Quick access to all modules

###  Category Management

 Create, Edit, Update, Delete Categories
 Display categories on UI

###  Subcategory Management

 Create Subcategories
 Link Subcategories with Categories
 Edit / Delete Subcategories
 Display Subcategories under Categories

###  Brand Management

 Add, Edit, Delete Brands
 Show brand listing in UI

###  Product Management

 Create Products
 Upload Multiple Product Images
 Edit / Update Product Details
 Delete Products
 Display Products on UI

###  Image Handling

 Upload images for Categories and Products
 Update and Delete images

###  UI & User Interaction

 Data listing with clean UI
 Edit/Delete actions directly from UI
 Success & Error Messages (Flash Messages)

###  AJAX Functionality

 Asynchronous requests using jQuery AJAX
 Smooth UI updates without full page reload

###  JSON Data Handling

 Data transfer using JSON format
 Efficient frontend-backend communication
